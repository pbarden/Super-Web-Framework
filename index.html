Text colors:

There are several options for text colors, often used to emphasize errors, warnings, or other types of messages to grab the attention of the user. In order to not rip off naming conventions used in popular frameworks these have unique names.

In general, the class names should be set to emphasize meaning, such as 'error', 'inactive' or 'warning'. Since this is for a class project, I decided to use design 'flavors' instead.


"Tomato" (Red text)
"Leaf" (Green text)
"Wheat" (Yellow text)
"Berry" (Blue text)
"Stone" (Gray text)

Show me a code sample:
<span class="tomato">Tomato text</span>
<span class="leaf">Leaf text</span>
<span class="wheat">Wheat text</span>
<span class="berry">Berry text</span>
<span class="stone">Stone text</span>

Lazy loading:

Lazy loading has been applied to all images and background images through a custom javascript in order to reduce the initial page load time by rendering images after the rest of the page has been loaded.


Images are lazy loaded automatically, just include the url in both data for both src- and data-src- attributes. Images set as backgrounds just need an additional class and the data-bg attribute is set to the url.


Using the img tag:

Show me a code sample:
<img src="transparent.png" data-src="image.png">

Background images:


Show me a code sample:
<div class="lazybg" data-bg="image.png"><div>
